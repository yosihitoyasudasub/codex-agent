# UI: Next.js 15

## 概要
SSR / SSG / Server Actions / API Routes を備えたフルスタック React フレームワーク。
バックエンドロジックを TypeScript で Next.js 内に実装できる。

## 適した用途
- 一般的な Web アプリ（CRUD・管理画面・ダッシュボード）
- SEO が重要なサイト
- バックエンドを TypeScript で完結させたい場合

## 生成ファイル
- `Dockerfile` （プロジェクトルート）
- `docker-compose.yml` に `frontend` サービスを追加

## 開発サーバー
`localhost:3000`
